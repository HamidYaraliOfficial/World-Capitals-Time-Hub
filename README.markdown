# 🇺🇸 English

## Overview
World Capitals Time Hub is a modern, responsive, and feature-rich web application that displays live digital and analog clocks for world capitals. The platform includes real-time timezone updates, Gregorian, Persian (Jalali), and Islamic Hijri calendars, advanced search, multilingual support, favorites management, and an elegant user interface.

## Features

- Live digital clocks
- Live analog clocks
- World capitals database
- Gregorian calendar
- Persian (Jalali) calendar
- Islamic Hijri calendar
- Advanced capital and country search
- Region filtering
- Favorite capitals system
- Dark and light themes
- Responsive design
- Multilingual interface
- Interactive world map section
- Local timezone detection
- UTC and GMT clocks
- Modern glassmorphism UI
- No external backend required

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- IANA Timezone Database
- Local Storage API

## Installation

### Option 1: Open Directly
1. Download the project files.
2. Open `index.html` in any modern browser.

### Option 2: Run with VS Code Live Server
1. Install Visual Studio Code.
2. Install Live Server extension.
3. Open project folder.
4. Start Live Server.

- 区域筛选
- 收藏系统
- 深色与浅色主题
- 响应式设计
- 多语言支持
